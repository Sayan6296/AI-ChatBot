# 🤖   AI Chatbot (React + Core Python + OpenRouter)
A full-stack AI chatbot using React frontend and Core Python backend with OpenRouter API integration.

## 🔗 Live Demo

👉 https://ai-chat-bot-gules-iota.vercel.app

## 💻 GitHub Repository

👉 https://github.com/Sayan6296/AI-ChatBot

---

## 📌 About Project

This is a full-stack AI Chatbot built using **React (frontend)** and **Core Python (backend)** without using any frameworks like Flask or Django.

The chatbot integrates with **OpenRouter API** to generate AI responses.

---

## 🚀 Features

* 💬 Chat with AI
* ⚡ Fast React UI (Vite)
* 🧠 AI response using OpenRouter API
* 🔗 Frontend ↔ Backend API integration
* 🌐 Fully deployed (Vercel + Render)

---

## 🛠 Tech Stack

* **Frontend:** React, Vite
* **Backend:** Core Python (HTTP Server)
* **API:** OpenRouter AI API
* **Deployment:** Vercel (Frontend), Render (Backend)

---

## ⚙️ How It Works

1. User enters message in React UI
2. Request sent to backend (`/chat`)
3. Backend calls OpenRouter API
4. Response sent back to frontend

---

## 🧪 Run Locally

### ▶ Frontend

```bash
cd frontend
npm install
npm run dev
```

### ▶ Backend

```bash
cd backend
python server.py
```
